# Thumblify AI

Thumblify AI is a full-stack MERN application built for creators to generate and recreate YouTube-style thumbnails using AI. The platform focuses on providing a simple workflow for generating thumbnail concepts, managing previous creations, and exploring community-generated ideas.

The project uses Groq for prompt optimization and Pollinations AI for image generation. Users can generate thumbnails from a topic or recreate existing concepts with targeted modifications.

## Features

- AI-powered thumbnail generation
- Thumbnail recreate workflow
- JWT authentication and protected routes
- Credit-based generation system
- MongoDB storage for thumbnail history
- Community feed with public thumbnails
- Download support for generated images
- Responsive frontend built with React
